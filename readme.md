# Stepy

Android app for managing repeating actions and its steps for future time to complete estimation


## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform


## Installation

Install Stepy from [releases](https://github.com/mrdarip/Stepy/releases)
Or build it with Gradle

```bash
./gradlew build
```
